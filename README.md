###  Project installation

`git clone https://github.com/AnastasiaZelenkova/frontend-project-44.git`

`cd path/to/project/`

`make install`

### How to play?

`cd path/to/project/`

`brain-games`

There ara 5 avaliable games:

1. `brain-even`

2. `brain-calc`

3. `brain-prime`

4. `brain-gcd`

5. `brain-progression`
